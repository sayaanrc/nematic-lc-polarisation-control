# Electro-Optic Polarisation Control Using Nematic Liquid Crystals

This repository contains the Python code developed as part of my research project at **Raman Research Institute (RRI), Bangalore**.

The project was carried out under the supervision of **Prof. Arun Roy**, Liquid Crystal Lab, RRI.

---

### About the Project

- **Title:** Electro-Optic Polarisation Control Using Nematic Liquid Crystals  
- **Institution:** Raman Research Institute (RRI), Bangalore  
- **Supervisor:** Prof. Arun Roy  
- **Research Domain:** Optical Physics, Photonics, Liquid Crystals  

The objective was to control and analyse laser polarisation states using **nematic liquid crystal cells** and model the behaviour using **Jones calculus**.
